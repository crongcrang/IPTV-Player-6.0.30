คุณสมบัติของ IPTV Player

เวอร์ชัน:
ผู้พัฒนา: Son Tong

1. การจัดการช่องรายการ (Channel Management)

  • นำเข้าและส่งออกช่อง: รองรับการนำเข้าช่องจากไฟล์ .js, .m3u, .w3u, และ .json ผ่านเมนูหรือ URL และส่งออกเป็น .js หรือ .m3u รวมถึงข้อมูล DRM
  • จัดกลุ่มช่อง: แสดงช่องตามกลุ่ม เช่น "รายการโปรด" และเลือกดูได้จากรายการด้านซ้าย
  • แก้ไขช่อง: เปลี่ยนชื่อช่องและกลุ่มผ่านโหมดแก้ไข รวมถึงอัปโหลดโลโก้จากไฟล์หรือ URL
  • เรียงลำดับช่อง: เรียงจาก A-Z, Z-A หรือกลับสู่ลำดับเริ่มต้น และรองรับการลากและวางเพื่อจัดเรียง
  • ค้นหาช่อง: มีแถบค้นหาเพื่อกรองช่องตามชื่อ

2. การเล่นสื่อ (Media Playback)

  • รองรับสื่อหลากหลายรูปแบบ:
    • IPTV Streams: รองรับ .m3u8 และ .mpd พร้อมจัดการ DRM (เช่น ClearKey)
    • YouTube: เล่นวิดีโอจาก URL ผ่าน iframe embed
    • ไฟล์ในเครื่อง: รองรับ .mp4, .mkv, .avi, .mov, .flv
    • สตรีม URL: กรอก URL สตรีมเพื่อเล่นโดยตรง
  • เครื่องเล่น JW Player: ใช้ JW Player แสดงเมทาดาทา เช่น ชื่อ, Bitrate, Codec และระยะเวลา
  • การควบคุมการเล่น: ปุ่มหยุด/เล่น, เปลี่ยนช่อง, เพิ่มช่องโปรด, ลบช่อง, จับภาพหน้าจอ, ดาวน์โหลดวิดีโอ และควบคุมด้วยแป้นพิมพ์
  • โหมดเต็มหน้าจอ: ปรับขนาดเครื่องเล่นให้เต็มหน้าจอเมื่อขยายหน้าต่าง

3. การจัดการ Proxy

  • Proxy Server ในตัว: สร้าง HTTP Proxy ในเครื่องเพื่อจัดการสตรีม .m3u8
  • การรีเฟรช URL: รีเฟรช URL สตรีมเมื่อ token เปลี่ยนหรือหมดอายุ
  • หยุด/เริ่ม Proxy: หยุด Proxy เมื่อปิดเครื่องเล่นและเริ่มใหม่เมื่อเล่นช่องใหม่

4. การดาวน์โหลดวิดีโอ

  • ดาวน์โหลดสตรีม .m3u8: ใช้ FFmpeg ดาวน์โหลดเป็น .mp4 กำหนดระยะเวลาได้ พร้อมแสดงสถานะแบบเรียลไทม์
  • ดาวน์โหลดวิดีโอ YouTube: ใช้ yt-dlp เลือกคุณภาพ (เช่น 720p, 1080p) และบันทึกเป็น .mp4

5. การจัดการช่องโปรด (Favorites)

  • เพิ่ม/ลบช่องโปรด: เพิ่มช่องจากปุ่ม "F" บนเครื่องเล่น
  • นำเข้า/ส่งออกช่องโปรด: บันทึกและโหลดช่องโปรดในรูปแบบ .json
  • แสดงช่องโปรด: มีกลุ่ม "รายการโปรด" แยกในรายการกลุ่ม

6. อินเทอร์เฟซผู้ใช้ (User Interface)

  • มุมมองสองแบบ: Grid View (ตารางพร้อมรูปภาพ) และ List View (รายการแนวนอน)
  • การปรับแต่งธีมและภาษา: รองรับภาษาไทย/อังกฤษ และธีมมืด
  • แถบเลื่อนข่าว: แสดงข่าวสั้นจาก RSS Feed ที่ท้ายหน้าต่าง
  • การจับภาพหน้าจอ: จับภาพวิดีโอและบันทึกเป็น .png
  • Tooltip: แสดงคำอธิบายปุ่มเมื่อวางเมาส์ทับ

7. การอัปเดตและการสนับสนุน

  • ตรวจสอบการอัปเดต: ตรวจสอบและดาวน์โหลดอัปเดตจากเซิร์ฟเวอร์
  • หน้าต่างเกี่ยวกับ: แสดงข้อมูลเวอร์ชันและผู้พัฒนา (Son Tong)
  • การบริจาค: เชิญชวนบริจาคผ่าน PayPal ในหน้า Welcome

8. การจัดการไฟล์นำเข้า (Imported Files)

  • แสดงรายการไฟล์นำเข้า: ดูและเลือกไฟล์ช่องที่เคยนำเข้า
  • ลบไฟล์นำเข้า: ลบไฟล์และกลับไปใช้ไฟล์เริ่มต้น

9. คุณสมบัติเพิ่มเติม

  • Xtream code: เพิ่่มระบบ Xtream Code
  • การโหลดโฟลเดอร์: เลือกโฟลเดอร์ที่มีไฟล์มีเดียเพื่อเล่น
  • การจัดการข้อผิดพลาด: แสดงข้อความเมื่อโหลดสตรีมหรือไฟล์ล้มเหลว
  • บันทึกการตั้งค่า: ใช้ electron-store เก็บข้อมูล เช่น ช่องโปรด, ภาษา, และโหมดมุมมอง

10. การเพิ่มประสิทธิภาพและความปลอดภัย

  • Caching: ใช้ NodeCache เก็บข้อมูลชั่วคราว ลดการโหลดซ้ำ
  • การล็อกไฟล์: ใช้ proper-lockfile ป้องกันการเขียนไฟล์พร้อมกัน
  • ปิดการใช้งาน Hardware Acceleration: ป้องกันปัญหาความเข้ากันได้

สรุป: IPTV Player เป็นเครื่องมือครบครันสำหรับการรับชมและจัดการสื่อสตรีมมิ่ง เน้นการใช้งานง่าย รองรับหลายแหล่งสื่อ และมีฟีเจอร์ขั้นสูง เช่น การดาวน์โหลด, การจัดการ Proxy, และการแก้ไขช่อง เหมาะสำหรับผู้ใช้ที่ต้องการประสบการณ์ IPTV แบบส่วนตัวและปรับแต่งได้
